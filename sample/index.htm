How to ?

Include ResponsiveClass

First, include and init the ResponsiveClass.

<script type="text/javascript" src="ResponsiveClass.js"></script> <script type="text/javascript"> var rc = RC.make(); </script>

Define your HTML

Then, write your HTML and use the built-in formats (small, medium, large). Add the trigger class "rc" to activate.

<article class="rc myBaseArticle" class-small="firstVariant" class-medium="secondVariant" class-large="thirdVariant"> ... </article>
